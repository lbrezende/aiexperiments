Agent Teams — Claude Opus 4.6

Orquestrador de Agentes

Como 4 agentes especializados colaboram para construir a landing page perfeita, gerenciados pelo Opus 4.6.

0 Orquestrador Decompor + Gerenciar
🧠
Orquestrador Principal
Claude Opus 4.6 — Agent Teams
  • Recebe o briefing completo do produto
  • Decompoe a tarefa em subtarefas independentes
  • Identifica dependencias entre agentes
  • Lanca agentes em paralelo quando possivel
  • Gerencia o fluxo e valida outputs
Briefing + Requirements ↓
1 Fase 1 — Fundacao ⚡ Paralelo
🎯
UX Designer
Arquitetura de Informacao + Shaping
  • Frame: Define problema e outcome
  • Requirements: R0-R10 do que a pagina precisa
  • Shapes: 3 opcoes de layout (A, B, C)
  • Fit Check: Matriz decisoria R x S
  • Breadboard: Affordances UI + wiring
  • Output: shaping-doc.md completo
✍️
Copywriter
Persuasao + Narrativa de Venda
  • Headline: "Crie Seu Portfolio Completo de UX com IA"
  • Subtitulo: "Do zero. Ao vivo. Em um fim de semana."
  • Narrativa: 2 caminhos + Human First AI Design
  • Beneficios: 6 bullets com dados concretos
  • Autoridade: Bio + credenciais NNg
  • CTAs: "Quero Participar" + urgencia
Estrutura (UX) + Conteudo (Copy) ↓
2 Fase 2 — Design Visual ⏳ Sequencial
🎨
Visual Designer
Identidade Visual + UI Design
  • Paleta: Dark mode (#0a0a0f) + gradientes azul/roxo
  • Tipografia: Space Grotesk (headings) + Inter (body)
  • Componentes: Glassmorphism cards, CTAs animados
  • Animacoes: Float, shimmer, pulse, gradient-shift
  • Imagens: Prompts para geracao AI de cada visual
  • Spacing: Sistema de 8px grid, hierarquia visual
Design System completo ↓
3 Fase 3 — Implementacao ⏳ Sequencial
💻
Front-end Developer
HTML5 + CSS3 + Responsividade
  • CSS: Design tokens, reset, componentes, animacoes (~800 linhas)
  • HTML: 8 secoes semanticas com formularios (~450 linhas)
  • Responsivo: Breakpoints 480, 768, 1024, 1200px
  • Forms: Validacao HTML5 (nome, email, whatsapp)
  • Performance: Zero frameworks, CSS puro, preconnect fonts
  • Output: index.html + css/style.css prontos
Pagina implementada ↓
4 Fase 4 — Review ✅ Validacao
🔍
Review Final
Orquestrador valida coerencia
  • Verifica alinhamento entre estrutura UX e implementacao
  • Valida que todo conteudo do Copy esta no HTML
  • Confirma que design visual foi fielmente implementado
  • Testa responsividade em multiplos breakpoints
  • Verifica formularios e validacoes
  • Ajustes finais e entrega
Legenda — Agentes e Papeis
Orquestrador
Claude Opus 4.6 gerencia agent teams, decompoe tarefas e identifica dependencias
UX Designer
Define IA da pagina usando Shape Up: requirements, shapes, fit check, breadboard
Copywriter
Cria todo conteudo persuasivo: headlines, narrativa, beneficios, CTAs, social proof
Visual Designer
Define paleta, tipografia, componentes visuais, animacoes e sugestoes de imagem
Front-end Dev
Implementa tudo em HTML5 + CSS3 puro, responsivo, sem frameworks
